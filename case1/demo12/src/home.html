*以前的es2015已经过时了，现在用的是最新的babel-preset-env。
        1.首先安装依赖
        cnpm i -D webpack babel-core babel-preset-env babel-loader
        2.webpack.config.js配置
        {
            test:/\.js$/,
            use:{loader:'babel-loader'},
            exclude:/node_modules/ //不转换的文件夹，必须写。
        }
        3.根目录建立 .babelrc文件
        {
            "presets": [
                "env"
            ]
        }
        4.结束
